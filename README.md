# Money Manager Web Application

## 📌 Objective

This project is a simple Money Manager web application that helps users track their income and expenses. It allows users to add, view, and delete transactions while calculating the total balance.

---

## 🚀 Features

### ✅ Add Transaction

* Enter amount, date, type (income/expense), category, and description
* Stores data using LocalStorage

### ✅ View Transactions

* Displays all transactions in a table
* Shows date, type, category, description, and amount

### ✅ Delete Transaction

* Remove any transaction from the list

### ✅ Summary Section

* Total Income
* Total Expense
* Net Balance

---

## ⚙️ How It Works

* User inputs transaction details through a form
* Data is stored in LocalStorage
* JavaScript dynamically updates the transaction list
* Income and expenses are calculated automatically

1. Download or clone the repository
2. Open `index.html` in your browser
